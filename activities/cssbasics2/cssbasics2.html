Enagement Questions

1. List out the different ways to select an item.
-To select an item we can use Tag Names, ID's, Classes, and Descendants.

2. What is the difference between an ID and a Class?
-The main difference between an ID and a Class is that ID's can only be used once and Classes can be reused. A tag can use more than one Class.

3. What is the difference between a SPAN tag and a DIV tag?
-The difference between a DIV and a SPAN tag is that a SPAN tag can only be used inline and does not cause a line break whereas a DIV tag kind of puts everything under it in a block.

4. What is the difference between a DIV tag and HEADER, MAIN, FOOTER?
-The DIV tag is a generic tag used to group elements together. The Header tag is used to group header element together. The Main tag is contains the main content of a webpage. The Footer is the where we can information such as copy right notices and other links.

5. What are different ways to select a color?
-to select a color, we can use a generic color name or we can also use the RGB values. We can also use a hex code to choose or select a color.
