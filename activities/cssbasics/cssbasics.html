Engagement Questions

1. Explain the difference between inline, embedded, and external CSS.
Inline CSS are HTML Attributes that are used as part of the HTML tag itself. Embedded CSS are HTML Attributes that are located in the header of page and uses a selector to select parts of the page to be affected by a certain HTML Attribute. An External CSS uses an external style sheet,a separate text file and is linked to the html file to make style changes.

2. Explain why you should only use external CSS.
As per class discussion, we should use external CSS to promote organization in our coding. Having separate files for styling each part of our web page is good practice and makes our codes easier to understand and edit.

3. What is a selector used for?
A selector is used to target elements on our web pages that we wanted to style.
