Engagement Question

1. The form tag has an "action" attribute. What does it do?
-The action attribute specifies where to send the form-data when a form is submitted.

2. The form tag has a "method" attribute. What does it do?
-The method attribute specifies how to send form-data. Either 'Get' or 'Post'

3. What is the difference between a "POST" and a "GET"?
-GET is used to request data from a specified resource. POST is used to send data to a server to create/update a resource.
