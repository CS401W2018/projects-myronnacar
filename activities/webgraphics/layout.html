Engagement Questions

1. What is the difference between block tags and inline tags? Give examples of each one.
-Block tags starts on a new line and takes up the full width available while inline tags, as the name implies, display content within the same line as surrounding text. One example of a block tag is the header tag. One example of line tag is the anchor link tag.

2. What is normal flow in HTML?
-The normal flow in HTML is the way that block and inline elements are displayed on a page before any changes are made to their layout. Top to bottom, left to right.

3. What does float do? Why do you need to clear the float?
-It places an element on left or right side of its container allowing text and inline elements to wrap around it.

4. What do absolute, fixed, and relative do?
-Absolute - the element is positioned absolutely to it first position. -Fixed - the element is positioned related to the browser window. It does not move, hence it's fixed. Relative - the element is positioned relative to its normal position.

5. How can you make nice navigation buttons for mobile devices?
-We are able to make nice navigation buttons for mobile devices by placing the link under a UL tag.
