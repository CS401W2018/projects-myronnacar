Engagement Questions

1. Why do we need a viewport meta tag?

The viewport meta tag is important because it gives the browser instructions on how to control the page;s dimensions and scaling.

2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

the last one would take priority. Both of them would be executed but the last tag would be the one that would be showing in the webpage.

3. How do you change the CSS for a web page displayed on a phone?

By use the media query we are able to edit the CSS that would display on a phone.
