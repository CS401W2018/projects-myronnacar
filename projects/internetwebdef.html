Internet

The Internet is a global network that connects billions of devices and allows users to communicate and access information. Through the internet, we are able to use email, instant messaging, video conferencing, and more.

Web

The Web is a system of interconnected web pages that are accessible through the internet.

How are the Internet and Web different?

While the Internet is a global network that connects a vast network of computers around the world, the Web is a system of interconnected web pages accessible through the internet. I think of it like this, the Internet is like the roads and highways that connects us from place to place and the Web is like the cars that run through those roads and highways.

ASCII and Unicode

Both the ASCII and the Unicode are character encoding standards that are used to represent computer language to the English language.

What is UTF-8?

The UTF-8 is a character encoding standard that translates Unicode characters into a sequence of bytes. The UTF-8 is preferable because of its backward compatibility with ASCII and it uses less memory.

URL

URL stands for Uniform Resource Locator. Simply put, a URL is a location or address of a unique resource found on the internet. It refers to a specific page on a certain website.

Domain Name

A Domain Name is the name of a website. For example, "google.com" is the domain name for the Google website.

HTTP

The HTTP is a set of rules that allows web clients and servers to be able to communicate with each other.

HTTPS

The HTTPS is a secure version of the HTTP. It is still a set of rules that allows communication web clients and servers however, HTTPS encrypts the data sent to protect both the connection and the data.

IP Address

An IP Address is a unique string of characters that identifies each computer or device that communicates over the internet or a network.

DNS

A DNS is a system that translates domain names into IP addresses. I think of DNS like a phone book that translates name into a phone number on your phone. DNS does the exact the same except it translates a Domain Name into an IP Address.

W3C Recommendations

The WC3 is an international non-profit organization that develops standards or guidelines that promotes consistency and interoperability on the web. It publishes recommendations that are considered to be web standards. These recommendations help developers and organization create reliable, consistent, and accessible web pages that are compatible across different browsers.

XML, XHTML, HTML, HTML5

The XML is primarily designed to store and transport data. It focuses on the structure of the data rather than how it's displayed. It is strictly well-formed and case-sensitive.

The XHTML is a hybrid between HTML and XML. It brings the strictness of XML to HTML for web documents. It is a little less flexible that HTML due to its strict rules. It is focused on both structure and presentation, like HTML

The HTML is the standard language for creating web pages and web applications. It focuses on the structure of web content. It has looser syntax compared to XML and XHTML.

The HTML5 is the latest and most poweful version of HTML. It is designed for modern web development and has introduced new features for multimedia, API's, and mobile friendliness. Its syntax is similar to HTML but has enhanced semantics that provide better structure to web page content.
